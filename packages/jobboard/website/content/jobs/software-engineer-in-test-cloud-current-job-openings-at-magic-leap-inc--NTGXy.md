## Job Description

As a Quality Engineer in our Cloud organization, you will have the opportunity to use cutting edge web technologies to develop automated tests for Magic Leap’s Platforms and APIs. The ideal candidate will have experience developing tests for responsive modern web interfaces consuming REST based web services, remarkable problem-solving skills, attention to detail, and a passion for testing. 

## Responsibilities

*   Develop automated tests using Behavior Driven Development processes and testing frameworks
*   Automate API tests and collaborate with Sr. Test Engineers on performance testing
*   Develops tools, frameworks, and systems to increase developer productivity
*   Work closely with designers, project managers, and back-end engineers 
*   Work closely with technical leaders and other software engineers in a collaborative environment to deliver next-generation web component 
*   Plays an instrumental role in the organization’s BDD transformation, acts as a change agent, and shows strong leadership skills with a positive and collaborative attitude
*   Learn best practices from senior and technical leadership team members
*   Embrace emerging web technology standards

## Qualifications

*   Experience developing tests for RESTful services using JavaScript, NodeJS, Java or GoLang
*   Experience with AWS, GCP, or Azure
*   Experience using at least one of the following:  Selenium, Mocha, Protractor, Serenity, or Cucumber
*   Knowledge of functional programming principles.
*   Experience testing production-quality web applications that consume REST services
*   Understanding of Test-Driven and/or Behavior-Driven Development
*   Results-oriented and deadline-driven
