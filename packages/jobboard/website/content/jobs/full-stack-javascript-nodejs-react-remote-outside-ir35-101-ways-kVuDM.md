101 ways is a digital consultancy that helps great clients do great things, solve challenging problems and build great digital products.

We're ramping up on a number of engagements and need both senior and lead full-stack JavaScript ( react/node ) and backend NodeJS developers to start in June and July. The projects available are all product-focused, a mix of both greenfield ramp-ups and joining existing 101 engagements, where we already have teams of engineers doing great work.

## Technologies & environment

*   JavaScript / Typescript
*   NodeJS
*   GraphQL
*   React with Redux
*   AWS / Serverless / Azure
*   Kubernetes
*   Cypress
*   Agile - XP, TDD, pair programming, CI/CD etc

We’re building full-stack delivery teams, so we’re open to hearing from front-end engineers, full-stack developers, and backend engineers that are open to and capable of working across the stack. Most importantly we value collaboration, commitment to quality, and a genuine belief in agility.

## Interview process

*   Recruiter introduction to discuss projects in more detail (max 30 mins)
*   Pair programming exercise (1 hour)
*   Meet a director (1 hour)

**UK-based applicants only**
